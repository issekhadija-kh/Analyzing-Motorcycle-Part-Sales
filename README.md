# Analyzing-Motorcycle-Part-Sales

This Project analyzes wholesale net revenue performance across different product lines, months and warehouses. The goal is to understand how much net revenue each product line generated per month and per warehouse for wholesale clients.

The objective is to calculate the net revenue generated by each product line for wholesale clients, broken down by month and warehouse, to identifiy revenue patterns and performnce trends.

## Expected output:
Creat a query named `'revenue_by_product_line'` that returns the following columns:

• `'product_line'`: Type of product ordered.

• `'month'`: Month name: 'June', 'July' or 'August'. 

• `'warehouse'`: warehouse location.

• `'net_revenue'`: Sum of `'total'` and `'payment_fee'`.

The results should be sorted by product_line and month, followed by net_revenue in descending order.
